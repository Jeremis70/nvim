# nvim config

Personal Neovim configuration.

## Structure

```
init.lua                  # entry point
lua/
  config/
    lazy.lua              # lazy.nvim bootstrap & setup
  plugins/                # plugin specs (one file per plugin/group)
    dankcolors.lua        # colorscheme + live hot-reload watcher
```

## Plugin Manager

Uses [lazy.nvim](https://github.com/folke/lazy.nvim). Run `:Lazy` to open the plugin UI.

## Colorscheme

Built on [base16-nvim](https://github.com/RRethy/base16-nvim) with a custom palette generated by [matugen](https://github.com/InioX/matugen) (Material You color generator).

`dankcolors.lua` watches itself via `uv.new_fs_event()` and hot-reloads the colorscheme on save, no Neovim restart needed. Matugen writes directly to this file when regenerating colors. Made for Dank Material Shell but should be reusable in other setups.
